# yp-link README

因为yp项目组规范要求。分支名必须是 特征/飞书id-name且commit的时候也必须带上飞书项目id，所以为了
1. 方便开发commit，便捷复制飞书id；
2. 方便打开项目文档。不需要特意去找对应的飞书项目，直接从编辑器打开文档

## Features
1. 复制分支名称
2. 复制飞书ID
3. 打开飞书项目文档

## Requirements
1. 安装拓展插件。目前用vsix，未发布到插件市场
2. run `YP-LINK: Show Branch And FeishuId` command
